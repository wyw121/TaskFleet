# 用户使用指南

## 快速入门

### 系统要求
- Windows 10/11 操作系统
- 至少4GB可用内存
- 1GB可用磁盘空间
- USB接口用于连接Android设备

### 安装步骤

1. **下载安装包**
   - 从官方渠道下载Flow_Farm安装包
   - 解压到任意目录

2. **运行程序**
   - 双击`Flow_Farm.exe`启动程序
   - 首次运行会进行环境检测

3. **设备驱动安装**
   - 程序会自动检测ADB环境
   - 如需要会提示安装相关驱动

## 设备配置

### Android设备准备

1. **开启开发者选项**
   ```
   设置 → 关于手机 → 版本号(连击7次)
   ```

2. **启用USB调试**
   ```
   设置 → 开发者选项 → USB调试(开启)
   ```

3. **连接确认**
   - 用USB线连接设备到电脑
   - 在设备上允许USB调试授权

### 设备管理界面

#### 添加设备
1. 点击"扫描设备"按钮
2. 从检测到的设备列表中选择
3. 点击"添加设备"完成配置

#### 设备状态监控
- 🟢 绿色: 设备在线且可用
- 🟡 黄色: 设备连接但应用未就绪  
- 🔴 红色: 设备离线或异常
- ⚪ 灰色: 设备未连接

#### 设备操作
- **设备信息**: 查看设备型号、系统版本等
- **应用管理**: 安装/卸载目标应用
- **截屏功能**: 实时查看设备屏幕
- **日志查看**: 设备操作日志记录

## 平台操作指南

### 小红书自动化

#### 准备工作
1. 确保设备已安装小红书APP
2. 手动登录小红书账号
3. 确认账号状态正常

#### 关注操作
1. **设置关注参数**
   - 目标关注数量: 建议每次20-50个
   - 操作间隔: 建议3-8秒随机间隔
   - 执行时间: 设置开始和结束时间

2. **启动关注任务**
   - 选择目标设备
   - 点击"开始关注"按钮
   - 实时监控执行进度

3. **监控执行状态**
   - 查看实时操作日志
   - 监控关注成功率
   - 观察异常情况处理

#### 安全建议
- ⚠️ 不要连续长时间运行
- ⚠️ 避免在同一时间大量操作
- ⚠️ 定期检查账号状态
- ⚠️ 遵守平台使用规范

## 任务管理

### 创建任务

1. **任务基本信息**
   ```
   任务名称: 小红书关注任务_20231201
   目标平台: 小红书
   执行设备: 选择已配置的设备
   ```

2. **任务参数设置**
   ```
   操作类型: 关注用户
   目标数量: 50个用户
   执行间隔: 3-8秒随机
   失败重试: 3次
   ```

3. **时间安排**
   ```
   立即执行: 创建后立即开始
   定时执行: 设置开始时间
   重复执行: 设置重复规则
   ```

### 任务监控

#### 实时状态面板
- **执行进度**: 当前任务完成百分比
- **成功率**: 操作成功/总操作次数
- **执行速度**: 每分钟操作次数
- **剩余时间**: 预计完成时间

#### 操作日志
```
[14:30:25] 开始执行关注任务
[14:30:28] 成功关注用户: user123
[14:30:35] 成功关注用户: user456  
[14:30:40] 检测到验证码，暂停3秒
[14:30:45] 继续执行任务
```

### 任务控制

#### 基本操作
- **暂停任务**: 临时停止执行，可恢复
- **停止任务**: 完全停止，不可恢复
- **重启任务**: 从头开始执行
- **查看详情**: 显示任务详细信息

#### 紧急停止
- 快捷键: `Ctrl + Q`
- 按钮: 红色"紧急停止"按钮
- 适用场景: 发现异常时立即停止所有操作

## 常见问题解答

### 设备连接问题

**Q: 设备无法被识别怎么办？**
A: 
1. 检查USB调试是否开启
2. 重新安装设备驱动
3. 尝试更换USB数据线
4. 重启设备和程序

**Q: 设备连接后显示离线？**
A:
1. 在设备上重新授权USB调试
2. 检查设备是否锁屏
3. 重启ADB服务
4. 确认设备系统版本兼容性

### 任务执行问题

**Q: 任务执行失败率很高？**
A:
1. 降低操作频率
2. 增加操作间隔时间
3. 检查目标应用是否正常
4. 验证网络连接状态

**Q: 程序运行一段时间后卡死？**
A:
1. 重启程序和设备
2. 检查系统内存使用情况
3. 清理临时文件
4. 更新到最新版本

### 权限和安全问题

**Q: 忘记管理员密码怎么办？**
A:
1. 联系系统管理员重置
2. 使用密码找回功能
3. 提供身份验证信息
4. 必要时重新安装系统

**Q: 如何保证操作安全性？**
A:
1. 严格控制操作频率
2. 定期检查账号状态
3. 遵守平台使用规范
4. 及时更新安全补丁
