### 🏗️ 技术架构

TaskFleet采用现代化的三层架构：

```
┌─────────────────────────────────────────────────────────┐
│                  TaskFleet 任务执行专家                    │
├─────────────────────────────────────────────────────────┤
│  📊 管理端 (Web)     │  💻 员工端 (Desktop)              │
│  React + TypeScript  │  Tauri + Rust                    │
├─────────────────────────────────────────────────────────┤
│              🔧 后端服务 (Rust + Axum)                    │
│           任务分发 • 进度监控 • 数据统计                    │
└─────────────────────────────────────────────────────────┘
```

## 🔍 项目特色

### 核心功能
- ✅ **任务分发** - 批量导入、智能分配
- ✅ **进度监控** - 实时状态、完成率统计
- ✅ **数据分析** - 员工效率、趋势分析

### 技术特点
- 🦀 **Rust后端** - 高性能、内存安全
- ⚛️ **React前端** - 现代化Web界面
- 🖥️ **Tauri客户端** - 原生桌面体验
- 📊 **实时通信** - WebSocket + RESTful API
